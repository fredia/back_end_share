# 项目介绍
本项目（综合性资源共享平台后端）是毕设项目（基于c++的web轻量级框架的设计与实现）的后台部分，毕设的重点部分在与web框架的设计与实现，综合性资源共享平台是对该框架可用性的一次实验验证。

主要工作如下：
- web框架
    - mysql-orm
    - sneeze(http服务器)
- 基于web框架的综合性资源共享平台(RestFul,前后端分离)
    - 后端(MVC三层架构)
    - 前端(VUE)
